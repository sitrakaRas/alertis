Foire aux questions pour les formations incendie :

Les formations incendie sont-elles obligatoires ?

Oui car le code du travail et les commissions de sécurité rendent obligatoire les formations incendie dans les établissements recevant du public (ERP) ainsi que dans les entreprises. D’un point de vue assurance, le référentiel APSAD demande que des employés soient formés EPI et ESI
